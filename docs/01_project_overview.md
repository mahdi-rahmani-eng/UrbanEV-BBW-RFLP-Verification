# Project Overview

This project develops a university-level lightweight MBSE/RFLP workflow for a low-speed urban EV.

The first part compares three vehicle concepts using MATLAB-based range estimation, braking-distance analysis, and normalized decision scoring.

The selected concept is then extended into concept-level brake-by-wire emergency brake-request logic. The project includes requirements, RFLP decomposition, actuator concept selection, simplified scenario inputs, and MATLAB-based verification.

The verification uses stopping distance, TTC, road-friction, and actuator-delay assumptions. Results are linked back to requirements through a traceability matrix.
